# Shared skills

Team-wide skills live here as canonical source material. Install or copy selected skills into an agent's native Hermes skill directory when that agent should use them:

```text
agents/<agent>/home/skills/
```

Suggested categories:

- `team/` — operating rhythm, routing, collaboration protocols
- `product/` — PRDs, strategy, roadmap methods
- `research/` — market, customer, competitor research
- `engineering/` — architecture, code review, delivery playbooks
- `design/` — UX critique, prototypes, brand systems
- `growth/` — GTM, campaigns, analytics
- `finance/` — forecasting, pricing, ops
- `risk/` — legal, compliance, security review
